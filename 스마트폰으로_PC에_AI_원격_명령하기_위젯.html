클로드 코드 스마트폰으로 PC에 AI 원격 명령하기
지금부터 설정하면 회사 밖 어디서든 스마트폰으로 내 PC의 Claude에게 업무를 지시할 수 있어요.
방법 A(코워크 디스패치)는 앱 설치만으로 바로 가능하고,
방법 B(텔레그램 채널)는 터미널을 조금 다루지만 더 자유롭게 활용할 수 있습니다.
이 가이드를 순서대로 따라하면 처음이어도 완성할 수 있어요.
사전 준비
💻
PC 환경 확인
PC에 Claude Code 설치가 되어 있어야 해요.
설치 여부 모르면 IT팀에 문의하세요.

PC는 실습 내내 절전 해제 상태로 켜두세요.
📱
스마트폰 앱 설치
텔레그램 앱을 미리 설치해두세요.
iOS → App Store에서 "Telegram" 검색
Android → Play Store에서 "Telegram" 검색

PC와 스마트폰 모두 같은 Claude 계정으로 로그인 확인.
단계별 실습
1
[방법 A] 코워크 디스패치 — 앱으로 바로 연결
  • PC에서 claude.ai 접속 → 상단 메뉴 "Cowork" 탭 클릭
  • 왼쪽 사이드바에서 "Dispatch" 선택
  • "+ New Dispatch" 클릭 → 새 디스패치 생성
  • 스마트폰에서 Claude 앱 열기 → 하단 Cowork 탭 진입
  • "이 PC에 연결" 버튼 탭 → 화면에 연결 완료 표시 확인
✔ 성공 확인: 스마트폰에서 "안녕이라고 말해줘"라고 입력했을 때 PC 터미널에서 Claude가 응답하면 완료입니다.
2
[방법 B] 텔레그램 봇 만들기 (약 5분)
  • 스마트폰 텔레그램 검색창에 @BotFather 입력 → 파란 체크 계정 선택
  • 채팅 시작 → /newbot 입력 후 전송
  • 봇 이름 입력 (예: 우리팀AI) → 전송
  • 봇 사용자명 입력 → 반드시 _bot으로 끝나야 해요 (예: ourteamai_bot)
  • BotFather가 보내준 토큰(긴 숫자:문자 조합)을 복사해서 메모장에 저장
⚠️ 토큰은 절대 다른 사람과 공유하지 마세요. 이 토큰으로 내 PC에 원격 접근이 가능해집니다.
3
[방법 B] 클로드 코드와 텔레그램 연결하기
  • PC에서 터미널 열기
    Mac: Cmd + Space → "terminal" 입력 → Enter
    Windows: Win 키 → "cmd" 입력 → Enter
  • 아래 명령어를 복사해서 붙여넣기 후 Enter
💻 터미널 명령어
claude channel add telegram
  • "봇 토큰을 입력하세요" 메시지 → 2단계에서 저장한 토큰 붙여넣기
  • 텔레그램에서 내 봇에게 아무 메시지 전송 (예: "안녕")
  • 터미널에 채팅 ID 숫자가 표시됨 → 그대로 입력 후 Enter
✔ 성공 확인: 터미널에 "채널 연결 완료" 메시지가 표시되면 완료입니다.
4
첫 원격 명령 테스트해보기
  • PC는 켜둔 채로 → 스마트폰에서 텔레그램 봇에 메시지 전송
  • 잠시 기다리면 PC에서 Claude가 자동 실행되고 결과가 생성됩니다
  • 결과물은 지정된 폴더에 자동 저장됩니다
💬 테스트 명령어 예시
바탕화면에 test.txt 파일 만들고 '원격 테스트 성공'이라고 써줘
💼 실제 업무 활용 예시
퇴근 후: "오늘 회의록 정리해서 메일 초안 써줘" 출퇴근 중: "이 데이터 분석해서 보고서 만들어줘" 주말에: "다음 주 발표자료 초안 잡아줘"
5
24시간 활용을 위한 PC 절전 해제 설정
  • Mac: 시스템 설정 → 배터리 → "절전 모드 시작 안 함" 체크
  • Windows: 제어판 → 전원 옵션 → "절전 안 함"으로 변경
  • Claude Code 자동 실행 등록이 필요하면 IT팀에 요청하세요
✔ 이 설정까지 완료하면 퇴근 후·주말에도 스마트폰 하나로 PC에 업무를 지시할 수 있습니다.
