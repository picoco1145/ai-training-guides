# 가이드 일괄 생성

## Step 1: 주제 목록 수집
사용자에게 여러 가이드 주제를 한 번에 입력받는다 (리스트, 번호 매기기 등).

## Step 2: 일괄 기획
planner 에이전트가 각 주제의 기획서를 순차 작성.
모든 기획서를 사용자에게 한 번에 제시:

```
📋 아래 {N}개 과제를 실습 가이드로 만들겠습니다:

1. {제목 1} — {도구} / {대상}
2. {제목 2} — {도구} / {대상}
3. {제목 3} — {도구} / {대상}

수정/삭제할 항목이 있으면 번호로 말씀해주세요.
"전체 진행" 하시면 순차 생성을 시작합니다.
```

## Step 3: ★ 사용자 점검 ★ (필수)
전체 목록 승인 후에만 진행.

## Step 4: 순차 생성
각 가이드를 guide-create 스킬의 Step 4~7로 순차 처리.
파일 충돌 방지를 위해 병렬 아닌 **순차** 실행.

## Step 5: 완료 리포트
```
✅ 일괄 생성 완료!

성공: {N}개 | 실패: {M}개

번호  제목                  상태      Notion URL
---  ---                   ---      ---
23   엑셀 매크로 기초         배포완료   https://...
24   코덱스 실습             배포완료   https://...
25   슬라이드 마스터          검수실패   (수정 필요)
```

## 주의사항
- Notion 파일 업로드 시한(1시간) 주의: 이미지 업로드 → 블록 첨부를 가이드별로 묶어서 처리
- 가이드당 이미지 3~8장 × 여러 가이드 → API 비용 고려
