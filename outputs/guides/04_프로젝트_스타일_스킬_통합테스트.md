# 프로젝트+스타일+스킬 3종 세트 통합 테스트

> 🏷 태그: Claude · 자동화
> 👥 대상: 과제 1~3 완료자
> ⏱ 소요시간: 20분

## 💡 소개
과제 1~3에서 **프로젝트(맥락)**, **커스텀 스타일(말투)**, **스킬(절차)** 을 각각 만들었습니다. 이 세 가지를 동시에 켜면 클로드는 "내가 어떤 사람인지(프로젝트) + 어떻게 쓰는지(스타일) + 어떤 순서로 쓰는지(스킬)"를 한 번에 알고 글을 씁니다. 주제만 던지면 나처럼 쓴 완성 글이 나오는지 직접 확인합니다.

## 📦 사전 준비
- 과제 1: 클로드 프로젝트 생성 완료
- 과제 2: 커스텀 스타일 저장 완료
- 과제 3: 스킬 파일 프로젝트에 업로드 완료

---

## 📚 단계별 실습

### 1️⃣ 3가지가 모두 세팅됐는지 체크하기

과제 1에서 만든 프로젝트로 이동합니다.

프로젝트 설정 화면에서 아래 세 가지를 확인합니다:

| 항목 | 확인 포인트 | 상태 |
|------|------------|------|
| **Instructions** | 글쓰기 맥락 지침이 입력되어 있는가 | ☐ |
| **Knowledge** | 내 글 샘플이 업로드되어 있는가 | ☐ |
| **Skills** | 과제 3에서 만든 .md 파일이 있는가 | ☐ |

✅ 확인: 세 항목 모두 체크되면 다음 단계로 넘어갑니다.

### 2️⃣ 스타일 선택 후 새 대화 시작하기

프로젝트 안에서 **"New chat"** 버튼을 클릭합니다.

채팅창 옆의 스타일 드롭다운에서 **과제 2에서 만든 내 커스텀 스타일**을 선택합니다.

<!-- CAPTURE: url=https://claude.ai area=full annotate=arrow(900,700,"커스텀 스타일 선택") -->

✅ 확인: 스타일 드롭다운에 내 스타일 이름이 선택된 상태이면 성공입니다.

---

### 3️⃣ 주제만 던져서 글 자동 생성하기

채팅창에 딱 이것만 입력합니다:

```
"2026년 AI 글쓰기 자동화 트렌드 블로그 글 써줘"
```

긴 설명 없이 주제만 입력합니다. 클로드가 알아서 처리하는지 봅니다.

<!-- CAPTURE: url=https://claude.ai area=full annotate=highlight(.streaming-response,"클로드가 생성하는 글") -->

클로드가 응답을 시작하면 아래를 체크합니다:

| 체크 항목 | 확인 방법 |
|----------|----------|
| 스킬이 발동했는가 | 먼저 제목 3개를 제안하는가? |
| 맥락이 반영됐는가 | 내 블로그 주제에 맞는 내용인가? |
| 말투가 맞는가 | 내 기존 글과 비슷한 문체인가? |

✅ 확인: 제목 제안 → 선택 → 본문 작성 순서로 진행되고, 말투가 자연스러우면 성공입니다.

---

### 4️⃣ 3종 세트 효과 비교하기

이번에는 일반 채팅(프로젝트 밖)에서 똑같은 질문을 해봅니다.

왼쪽 사이드바에서 **"New chat"** (프로젝트 밖)을 클릭합니다.

<!-- CAPTURE: url=https://claude.ai area=full annotate=arrow(50,200,"사이드바의 New chat (프로젝트 밖)") -->

똑같이 입력합니다:
```
"2026년 AI 글쓰기 자동화 트렌드 블로그 글 써줘"
```

두 결과를 비교합니다:

<!-- GENERATE: prompt="side by side comparison diagram: left box labeled '일반 채팅 (3종 세트 없음)' showing generic output, right box labeled '프로젝트+스타일+스킬' showing personalized output with arrows pointing to differences, clean flat design" style=diagram -->

| | 일반 채팅 | 3종 세트 |
|-|----------|---------|
| 말투 | 표준적, 딱딱함 | 내 말투 반영 |
| 구조 | 랜덤 | 스킬의 절차대로 |
| 맥락 | 없음 | 내 블로그 주제 반영 |

✅ 확인: 두 결과가 확연히 다르면, 3종 세트가 제대로 작동하고 있는 것입니다.

---

### 5️⃣ 미흡한 부분 수정하기

비교 결과 개선이 필요한 부분이 있다면 아래 체크리스트로 원인을 찾습니다:

| 증상 | 원인 | 해결 |
|------|------|------|
| 말투가 여전히 딱딱하다 | 스타일 지침이 부족 | 과제 2로 돌아가 스타일에 예시 문장 추가 |
| 스킬이 발동 안 된다 | description 키워드 불일치 | 스킬 파일 열어서 description 키워드 수정 후 재업로드 |
| 주제가 엉뚱하다 | 프로젝트 지침이 불명확 | 과제 1로 돌아가 Instructions에 블로그 주제 더 구체적으로 추가 |
| 구조가 엉망이다 | 스킬 절차 불명확 | 스킬 파일의 단계별 지침을 더 구체적으로 수정 |

✅ 확인: 수정 후 다시 테스트했을 때 원하는 글이 나오면 시스템 완성입니다.

## 💼 실무 팁

- **3종 세트는 한 번 세팅으로 계속 씁니다**: 한 번 잘 만들어두면 매번 설명 없이 주제만 던지면 됩니다.
- **챗GPT로 리서치, 클로드로 글쓰기**: ChatGPT는 최신 정보 검색과 데이터 분석이 강하고, 클로드는 긴 글 작성과 일관된 스타일 유지가 강합니다. 둘을 조합하세요.
- **스타일은 자주 테스트**: 새 글을 쓸 때마다 "이게 정말 내 말투인가?" 확인하고 아니면 스타일을 다듬어 나가세요.
