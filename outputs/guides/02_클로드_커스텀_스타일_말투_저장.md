# 클로드 커스텀 스타일로 내 말투 저장하기

> 🏷 태그: Claude · ChatGPT · 프롬프트 기법
> 👥 대상: 클로드 웹 초보자
> ⏱ 소요시간: 30분

## 💡 소개
클로드에는 **커스텀 스타일(Custom Styles)** 기능이 있습니다. 내가 평소 쓰는 글투, 문장 길이, 강조 표현 방식을 분석해서 저장해두면, 이후 어떤 주제로 글을 써달라고 해도 나처럼 쓴 글이 나옵니다. ChatGPT로 내 글 DNA를 먼저 뽑고, 그 결과를 클로드 스타일에 넣는 2단계 방법을 배웁니다.

## 📦 사전 준비
- [claude.ai](https://claude.ai) 계정
- [ChatGPT](https://chat.openai.com) 계정 (무료 가능)
- 내가 쓴 블로그/SNS 글 2~5편 (텍스트 복사 준비)

---

## 📚 단계별 실습

### 1️⃣ ChatGPT에서 내 글쓰기 DNA 분석하기

[chat.openai.com](https://chat.openai.com)에 접속합니다.

<!-- CAPTURE: url=https://chat.openai.com area=full annotate=highlight(.composer-parent,"채팅 입력창") -->

채팅창에 아래 프롬프트를 붙여넣고, 그 뒤에 **내 글 샘플을 2~5편** 이어서 붙여넣습니다:

```
첨부한 내 글 스타일을 완벽 분석하여 브랜드 보이스를 만들어줘.

1. 후킹 문구 스타일과 패턴
2. 도입부 구성 방식
3. 공감 표현 방법
4. 스토리텔링 기법
5. CTA(행동 유도) 표현 방식
6. 문장 길이와 구조적 특징
7. 자주 사용하는 표현과 어휘
8. 글의 전체적인 톤과 분위기

이 분석을 바탕으로:
- 나만의 글쓰기 DNA 추출
- 다른 주제에도 일관되게 적용 가능한 템플릿 생성

--- 내 글 시작 ---
[여기에 내 글 2~5편 붙여넣기]
```

✅ 확인: ChatGPT가 "후킹 문구 스타일", "문장 구조 특징" 등 항목별로 분석 결과를 출력하면 성공입니다.

---

### 2️⃣ 분석 결과를 클로드용 지침 프롬프트로 변환하기

ChatGPT 분석 결과 바로 아래에 이어서 다음을 입력합니다:

```
방금 분석한 내 글의 특징을 바탕으로, 주제와 핵심 내용만 입력하면
내 스타일의 블로그 글을 자동으로 작성해주는
클로드용 지침(컨텍스트 프롬프트)을 만들어줘.

조건:
- 내 글쓰기 스타일의 핵심 특징 포함
- 후킹+공감+CTA 구조 유지
- 2,000자 이상으로 구체적으로
- XML 태그 형식으로 작성
```

<!-- CAPTURE: url=https://chat.openai.com area=full annotate=highlight(.markdown,"ChatGPT 응답 영역") -->

ChatGPT가 긴 XML 형식의 지침을 출력합니다. 이 전체를 **드래그해서 복사**합니다 (Ctrl+A가 아닌 해당 응답만 선택).

✅ 확인: `<style>`, `<tone>` 같은 XML 태그가 포함된 긴 텍스트가 출력되면 성공입니다.

---

### 3️⃣ 클로드 커스텀 스타일 만들기

[claude.ai](https://claude.ai)로 이동합니다. 화면 오른쪽 아래 또는 채팅창 옆의 **스타일 선택** 드롭다운을 찾습니다.

<!-- CAPTURE: url=https://claude.ai area=full annotate=arrow(900,700,"스타일 선택 버튼") -->

드롭다운을 클릭하면 **"Normal"**, **"Concise"**, **"Explanatory"** 같은 기본 스타일 목록이 보입니다. 맨 아래의 **"Create & edit styles"** 또는 **"+ Create style"** 을 클릭합니다.

<!-- CAPTURE: url=https://claude.ai area=full annotate=highlight([data-testid="style-picker"],"스타일 드롭다운") -->

✅ 확인: 스타일 편집 화면(텍스트 입력창)이 열리면 성공입니다.

---

### 4️⃣ ChatGPT가 만든 지침 붙여넣고 저장하기

스타일 편집 화면의 텍스트 입력창에 2단계에서 복사한 내용을 **붙여넣기** 합니다.

<!-- CAPTURE: url=https://claude.ai area=full annotate=highlight(textarea,"지침 입력창") -->

스타일 이름을 입력합니다. 예: `내 블로그 스타일`, `세인투 스타일`

**"Save"** 또는 **"Create style"** 버튼을 클릭합니다.

✅ 확인: 스타일 드롭다운 목록에 내가 만든 스타일 이름이 나타나면 성공입니다.

---

### 5️⃣ 커스텀 스타일로 테스트 글 작성하기

새 대화를 시작합니다. 채팅창 옆의 스타일 드롭다운에서 방금 만든 **내 스타일**을 선택합니다.

<!-- CAPTURE: url=https://claude.ai area=full annotate=arrow(900,700,"내 스타일 선택") -->

아래처럼 간단하게만 입력합니다:

```
"클로드 vs ChatGPT 비교글 써줘. 1,000자 분량으로."
```

출력된 글을 내 기존 글과 비교해보세요.
- 문장이 비슷하게 짧은가?
- 같은 방식으로 독자에게 말을 거는가?
- 마무리에 행동 유도(CTA)가 있는가?

✅ 확인: 내가 직접 쓴 것처럼 어색하지 않은 글이 나오면 성공입니다.

---

## ⚠️ 자주 겪는 문제

- **스타일 메뉴가 안 보인다**: 커스텀 스타일은 **Pro 플랜** 전용입니다. 무료 플랜이면 스타일 선택 드롭다운이 없거나 기본 스타일만 보입니다. 이 경우 프로젝트 지침(Instructions)에 말투 지침을 직접 붙여넣는 방식으로 대체하세요.
- **"Create & edit styles" 버튼이 없다**: claude.ai UI가 업데이트되면 버튼 위치가 바뀔 수 있습니다. 채팅창 아래쪽의 아이콘들을 살펴보거나, 설정(⚙️) → 스타일 메뉴를 찾아보세요.

## 💼 실무 팁

- **샘플이 많을수록 정확합니다**: 글 샘플이 2편보다 5편일 때 분석 정밀도가 훨씬 높아집니다.
- **스타일은 주제별로**: 블로그용과 SNS용 말투가 다르다면 각각 스타일을 만드세요.
- **3개월에 한 번 업데이트**: 내 글투는 시간이 지나면서 바뀝니다. 최신 글로 스타일을 갱신하면 더 자연스럽습니다.
