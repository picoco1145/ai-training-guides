Computer Use Claude 자동화 Mac 설정
Claude 'Computer Use' 활성화하기
클로드 데스크톱 앱에서 AI가 마우스·키보드를 직접 조작할 수 있도록 'Computer Use' 기능을 활성화하는 첫 번째 실습입니다. 이 과정을 마치면 클로드에게 컴퓨터 작업을 맡길 수 있는 가장 기본적인 환경이 갖춰집니다.
사전 준비
🖥️
지원 기기 macOS (Windows는 추후 지원 예정)
✅
필수 앱 Claude Desktop App — 공식 홈페이지에서 다운로드
단계별 실습
1
데스크톱 앱 설정 진입
  • 클로드 앱 실행 후 Settings(설정) 메뉴로 이동
  • General(일반) 섹션 클릭
2
Computer Use 기능 활성화
  • 설정 항목 중 'Computer Use' 토글 스위치를 찾아 On 상태로 변경
  • 이 기능이 켜져 있어야 클로드가 화면을 캡처하고 클릭 이벤트를 발생시킬 수 있음
3
권한 승인 및 첫 명령 내리기
  • 대화창에 아래 테스트 프롬프트 입력:
내 컴퓨터의 제어권을 사용하여 '시스템 설정' 앱을 열어줘.
  • 보안 권한 팝업이 뜨면 Allow(허용) 클릭 → 클로드가 동작 시작
실무 팁
📱 모바일 승인
스마트폰의 클로드 앱으로 PC 작업 권한을 원격에서 승인할 수 있습니다.
🔒 개인정보 보호
클로드는 작업 중 주기적으로 화면을 캡처합니다. 민감한 정보가 열린 창은 명령 전 미리 닫아두세요.
⚡ 속도 최적화
초기 단계라 속도가 다소 느릴 수 있습니다. 복잡한 명령보다 단계별로 지시하면 정확도가 높아집니다.
