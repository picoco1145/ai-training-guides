Claude in PPT 오탈자 수정 AI 자동화
과제 3 — Claude in PPT로
오탈자 자동 검색/수정
PPT 추가기능 Claude in PPT를 설치하고, 슬라이드 전체의 오탈자를 AI가 자동으로 찾아 수정하는 기능을 실습합니다. 수십 장짜리 보고서도 프롬프트 한 줄로 전체 교정이 가능하며, 수정 전후를 비교해 최종 확인하는 과정까지 연습합니다.
사전 준비
🛠

준비물

  • 과제 1~2에서 만든 .pptx 파일
  • Claude.ai 계정 (무료 플랜 가능)
  • Claude in PPT 추가기능 설치 완료
✅

설치 확인

  • PPT 상단 [홈] 탭 또는 [추가기능] 탭에 "Claude" 아이콘 있으면 설치 완료
  • 없으면: [삽입] → [추가기능 가져오기] → "Claude" 검색 → 추가
  • 처음 실행 시 Claude 계정 로그인 필요
단계별 실습
1
오탈자가 포함된 실습용 슬라이드 만들기
  • 과제 1~2 파일 열기 → "본문" 레이아웃으로 슬라이드 3장 추가
  • 각 슬라이드에 의도적으로 오탈자 포함한 텍스트 입력
  • 예시 오탈자: "보고서를 작셩했습니다", "매출이 증가하였슴니다", "결과를 분석 하였다"
  • 띄어쓰기 오류도 포함: "사업계획서를작성합니다", "내용을 확인 해주세요"
💡 실습용 오탈자를 직접 만드는 이유: 수정 전/후를 눈으로 비교할 수 있어야 Claude가 제대로 동작했는지 확인이 됩니다.
2
Claude in PPT 실행하기
  • PPT 상단 탭에서 [Claude] 아이콘 클릭
  • 우측에 Claude 패널이 열림
  • 아직 로그인이 안 됐다면 [Sign in] 클릭 → Claude.ai 계정으로 로그인
  • 로그인 후 패널 상단에 대화 입력창이 보이면 준비 완료
💡 Claude in PPT는 현재 열려 있는 PPT 파일의 내용을 자동으로 읽습니다. 별도로 파일을 업로드할 필요가 없습니다.
3
오탈자 검색 프롬프트 입력하기
  • Claude 패널 입력창에 아래 프롬프트 입력:
  • "이 PPT의 모든 슬라이드에서 오탈자, 띄어쓰기 오류를 찾아서 목록으로 알려줘"
  • Enter 또는 전송 버튼 클릭
  • Claude가 슬라이드를 분석해 오류 목록을 응답으로 출력
  • 응답 내용 확인: 슬라이드 번호, 오류 텍스트, 수정 제안 형태로 나오면 정상
💡 응답이 영어로 나오면 프롬프트에 "한국어로 답변해줘"를 추가하세요.
4
오탈자 자동 수정 적용하기
  • Claude 응답 확인 후, 입력창에 다음 프롬프트 입력:
  • "찾아낸 오탈자를 모두 자동으로 수정해줘"
  • Claude가 각 슬라이드를 직접 수정하는 과정이 화면에 표시됨
  • 수정 완료 후 각 슬라이드로 이동해 변경된 내용 눈으로 확인
  • 원치 않는 수정이 있으면 Ctrl+Z로 해당 부분 되돌리기
💡 자동 수정 전에 파일을 다른 이름으로 저장(Ctrl+Shift+S)해두세요. 수정 결과가 마음에 안 들면 원본으로 되돌릴 수 있습니다.
5
수정 전/후 비교하고 최종 저장하기
  • 수정된 슬라이드를 처음부터 끝까지 슬라이드쇼로 검토: F5
  • 오탈자가 없어졌는지, 의미가 바뀐 부분은 없는지 확인
  • 추가 수정이 필요하면 Claude 패널에서 구체적으로 지시 (예: "3번 슬라이드 2번째 줄만 다시 확인해줘")
  • 이상 없으면 Ctrl+S로 최종 저장
💡 Claude는 문맥상 어색한 표현도 제안해줍니다. 오탈자 수정 외에 "문장을 더 자연스럽게 다듬어줘"도 시도해보세요.
실무 팁

수정 전 백업 필수

자동 수정 전에 반드시 다른 이름으로 저장하세요. AI 수정 결과가 예상과 다를 수 있습니다.

범위 지정 수정

특정 슬라이드만 수정하려면 "3번~5번 슬라이드만 확인해줘"처럼 범위를 지정하면 더 정확합니다.

문체 통일도 가능

"모든 문장을 합쇼체로 통일해줘" 같은 문체 교정도 동일한 방식으로 요청할 수 있습니다.
