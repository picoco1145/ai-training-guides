Claude in PPT 시각화 인포그래픽
과제 4 — 본문 내용 자동 시각화
텍스트로만 가득 찬 슬라이드를 Claude in PPT에게 넘겨, 도표·아이콘·인포그래픽으로 자동 변환하는 기능을 실습합니다. 시각화 전후를 직접 캡처해 비교하고, 프롬프트를 다르게 입력했을 때 결과가 어떻게 달라지는지도 확인합니다.
사전 준비
🛠

준비물

  • 과제 1~3에서 사용한 .pptx 파일
  • Claude in PPT 로그인 상태 유지
  • 화면 캡처 도구 (Windows: Win+Shift+S)
✅

실습용 텍스트 예시

  • 슬라이드에 아래 내용을 텍스트로만 입력해두기
  • 예: "1단계: 목표 설정 / 2단계: 전략 수립 / 3단계: 실행 / 4단계: 평가"
  • 또는: "매출 30% 증가 / 고객 만족도 95% / 신규 고객 200명 확보"
단계별 실습
1
시각화 전 슬라이드 캡처해두기
  • 텍스트만 있는 슬라이드로 이동
  • Win+Shift+S (Mac: Cmd+Shift+4) → 슬라이드 영역 드래그해서 캡처
  • 캡처 이미지를 바탕화면에 "시각화_전.png"로 저장
  • 이 캡처가 나중에 "전/후 비교"의 기준이 됩니다
💡 캡처 없이 진행하면 원래 상태로 되돌아가기 어렵습니다. 꼭 먼저 캡처하세요.
2
순서 흐름이 있는 텍스트 → 화살표 다이어그램으로 변환
  • Claude 패널 입력창에 프롬프트 입력:
  • "현재 슬라이드의 텍스트 내용을 순서 흐름을 보여주는 화살표 다이어그램으로 시각화해줘"
  • Claude가 도형과 화살표를 자동으로 삽입하는 것을 확인
  • 결과 슬라이드 캡처 → "시각화_후_흐름도.png"로 저장
  • 전/후 이미지를 나란히 놓고 비교
💡 결과가 마음에 들지 않으면 Ctrl+Z로 되돌리고 다시 시도하세요. 같은 프롬프트도 실행마다 결과가 조금씩 다를 수 있습니다.
3
수치 데이터 → 아이콘+숫자 인포그래픽으로 변환
  • 수치가 담긴 새 슬라이드로 이동 (예: 매출 30% 증가, 고객 만족도 95%)
  • Claude 패널에 프롬프트 입력:
  • "이 슬라이드의 수치 데이터를 아이콘과 숫자를 강조한 인포그래픽 스타일로 바꿔줘. 회사 색 테마를 유지해줘."
  • 결과 확인 후 캡처 → "시각화_후_인포그래픽.png"로 저장
💡 "회사 색 테마를 유지해줘"를 넣으면 과제 2에서 등록한 색상으로 시각화됩니다. 빼면 기본 색으로 생성됩니다.
4
프롬프트를 바꿔서 다른 스타일로 재시각화
  • Ctrl+Z로 직전 시각화 결과를 되돌리기
  • 이번엔 다른 스타일로 요청:
  • "이 슬라이드를 비교 표(테이블) 형식으로 시각화해줘"
  • 또는: "이 내용을 피라미드 다이어그램으로 만들어줘"
  • 결과를 캡처해서 앞의 결과와 비교 — 어떤 스타일이 내용 전달에 더 적합한지 판단
💡 "표", "원형차트", "막대그래프", "피라미드", "타임라인" 등 원하는 도표 종류를 직접 명시하면 더 정확한 결과가 나옵니다.
5
전/후 비교 캡처를 새 슬라이드에 정리하기
  • 새 슬라이드 추가 → 제목: "시각화 전/후 비교"
  • [삽입] → [그림] → "시각화_전.png" 삽입 → 좌측 배치
  • 동일하게 "시각화_후_흐름도.png" 또는 "시각화_후_인포그래픽.png" → 우측 배치
  • 각 이미지 아래 텍스트박스로 "전", "후" 레이블 추가
  • Ctrl+S 최종 저장
💡 전/후 비교 슬라이드를 만들면 보고서나 발표에서 "AI 활용 전후"를 직접 보여줄 수 있어 설득력이 높아집니다.
실무 팁

도표 종류 직접 명시

"시각화해줘"보다 "타임라인으로 만들어줘"처럼 구체적인 종류를 명시할수록 원하는 결과가 나옵니다.

색 테마 연동

프롬프트에 "회사 색 테마 유지"를 명시하면 과제 2에서 등록한 브랜드 컬러가 자동 반영됩니다.

원본 항상 백업

시각화 전 슬라이드를 별도 파일로 저장해두면 여러 스타일을 비교 실험할 때 원본을 잃지 않습니다.
