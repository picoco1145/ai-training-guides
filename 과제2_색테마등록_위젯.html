PPT 자동화 색 테마 슬라이드 마스터
과제 2 — 회사 색 테마 등록하기
회사(또는 본인) 브랜드 컬러를 슬라이드 마스터 색상 팔레트에 등록하고, .thmx 테마 파일로 저장하여 언제든 재사용할 수 있게 만들어봅니다. 색 테마를 한 번 등록하면 PPT 어디서나 클릭 한 번으로 회사 컬러를 적용할 수 있어, Claude in PPT가 슬라이드를 생성할 때도 자동으로 브랜드 색상이 유지됩니다.
사전 준비
🎨

준비물

  • 과제 1에서 저장한 .pptx 파일
  • 회사 메인 컬러 HEX 코드 (예: #1E3A8A)
  • 보조 컬러 HEX 코드 1~2개 (없으면 임의 생성)
✅

HEX 코드 찾는 법

  • 회사 홈페이지 로고 이미지를 캡처
  • Windows: Win+Shift+S 로 캡처 후 그림판에서 색 추출
  • 또는 구글에서 "color picker" 검색해 색상 코드 확인
단계별 실습
1
슬라이드 마스터에서 색 테마 편집 화면 열기
  • 과제 1 파일 열기 → [보기] → [슬라이드 마스터] 진입
  • 상단 [슬라이드 마스터] 탭에서 [색] 버튼 클릭
  • 드롭다운 목록 맨 아래 [색 사용자 지정...] 클릭
  • "새 테마 색 만들기" 창이 열림
💡 [색] 버튼이 안 보이면 PowerPoint 창을 넓게 늘려보세요. 창이 좁으면 일부 버튼이 숨겨집니다.
2
강조색(Accent) 컬러를 회사 색으로 변경하기
  • "새 테마 색 만들기" 창에서 [강조 1] 옆 색상 버튼 클릭
  • 색 선택 창 하단 [다른 색...] 클릭 → [사용자 지정] 탭 선택
  • HEX 코드 입력란에 회사 메인 컬러 코드 입력 (예: 1E3A8A)
  • 확인 → 강조 1 색이 변경된 것 확인
  • 동일 방법으로 [강조 2]에 보조 컬러 입력
💡 HEX 코드 입력 시 # 기호는 제외하고 숫자+영문만 입력하세요. 예: #1E3A8A → 1E3A8A
3
텍스트/배경 색도 브랜드에 맞게 조정하기
  • 같은 창에서 [텍스트/배경 - 어둡게 1] → 회사 메인 다크 컬러로 변경
  • [텍스트/배경 - 밝게 1] → 흰색(#FFFFFF) 또는 연한 배경색 유지
  • 오른쪽 미리보기에서 전체 색 조합 확인
  • 아래 [이름]란에 "회사명_테마" 형식으로 이름 입력 (예: ABC_테마)
  • [저장] 클릭
💡 텍스트/배경 색을 너무 강한 색으로 바꾸면 가독성이 떨어집니다. 어두운 색은 제목용, 밝은 색은 배경용으로 구분하세요.
4
색 테마가 적용됐는지 확인하기
  • [마스터 보기 닫기] 클릭해 일반 편집 화면으로 복귀
  • 슬라이드에 텍스트박스 삽입 → 텍스트 선택 → 글꼴 색 버튼 옆 드롭다운 클릭
  • 색상 팔레트에 방금 등록한 회사 컬러가 "테마 색" 영역에 표시되면 성공
  • 도형 삽입 후 [도형 채우기]에서도 동일하게 테마 색 확인
💡 테마 색으로 칠해진 요소는 나중에 테마를 바꿔도 자동으로 색이 업데이트됩니다. 일반 색을 쓰면 업데이트가 안 됩니다.
5
테마를 .thmx 파일로 저장해 재사용 가능하게 만들기
  • 상단 [디자인] 탭 클릭
  • 테마 목록 오른쪽 끝 [▼ 더보기] 클릭
  • 드롭다운 하단 [현재 테마 저장...] 클릭
  • 파일명 입력 (예: ABC회사_테마) → [저장]
  • 이후 어떤 PPT 파일에서든 [디자인] → [테마] → 목록에서 내 테마 선택 가능
💡 .thmx 파일은 기본적으로 C:\Users\[사용자명]\AppData\Roaming\Microsoft\Templates\Document Themes\ 에 저장됩니다. 팀원과 공유하려면 이 파일을 전달하세요.
실무 팁

HEX 코드 구하기

회사 홈페이지 로고를 캡처 후 구글 "color picker"에 붙여넣으면 HEX 코드를 바로 확인할 수 있습니다.

테마 색 vs 직접 색

반드시 테마 색으로 칠해야 나중에 테마 교체 시 자동 업데이트됩니다. 직접 선택한 색은 업데이트 안 됩니다.

팀원과 공유

.thmx 파일을 공유 드라이브에 올려두면 팀 전체가 동일한 색 테마를 적용할 수 있습니다.
