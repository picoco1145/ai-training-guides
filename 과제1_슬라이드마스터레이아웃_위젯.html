PPT 자동화 슬라이드 마스터 Claude in PPT
과제 1 — 슬라이드 마스터 레이아웃
직접 만들기
파워포인트의 슬라이드 마스터 기능을 직접 설정하여 회사 서식에 맞는 일관된 레이아웃을 만들어봅니다. 한 번 만들어두면 모든 슬라이드에 자동 적용되어 보고서 작성 시간을 크게 줄일 수 있습니다. 이후 과제에서 사용할 Claude in PPT가 슬라이드 마스터를 기반으로 동작하므로, 이 과제가 전체 실습의 기초가 됩니다.
사전 준비
🛠

준비물

  • Microsoft PowerPoint 2016 이상
  • 회사 로고 이미지 파일 (없으면 임의 사용)
  • 회사 메인 컬러 코드 (없으면 임의 색상 사용)
✅

사전 확인

  • PPT 새 파일을 열 수 있는지 확인
  • [보기] 메뉴가 상단에 보이는지 확인
  • 실습용 빈 .pptx 파일 미리 생성
단계별 실습
1
슬라이드 마스터 진입하기
  • PPT 상단 메뉴에서 [보기] 탭 클릭
  • 리본 메뉴 오른쪽 영역의 [슬라이드 마스터] 버튼 클릭
  • 화면이 바뀌면서 좌측 패널에 슬라이드 목록이 나타남
  • 좌측 패널 가장 위 큰 슬라이드가 '마스터 슬라이드' — 클릭해서 선택
  • 아래 작은 슬라이드들은 '레이아웃 슬라이드' — 각각 다른 형태의 레이아웃
💡 마스터 슬라이드에서 수정한 내용(폰트, 로고 등)은 모든 레이아웃에 자동 반영됩니다.
2
레이아웃 1: 표지 슬라이드 만들기
  • 좌측 패널에서 마스터 바로 아래 첫 번째 레이아웃 슬라이드 선택
  • 우클릭 → [이름 바꾸기] → "표지"로 이름 변경
  • 중앙 제목 텍스트박스 클릭 → 폰트 크기 36pt, 굵게, 색상 설정
  • [삽입] → [그림] → 회사 로고 이미지 삽입 → 우측 상단에 배치 후 위치 잠금
  • 배경 색상 변경: 우클릭 → [배경 서식] → 단색으로 회사 메인 컬러 적용
💡 로고 위치 잠금: 로고 선택 → [도형 서식] → [크기 및 위치] → "위치 고정" 체크
3
레이아웃 2: 본문(텍스트) 슬라이드 만들기
  • 좌측 패널에서 두 번째 레이아웃 선택 → 이름 "본문"으로 변경
  • 상단 제목 플레이스홀더 선택 → 높이 1.5cm, 상단 여백 1cm로 조정
  • 본문 플레이스홀더 선택 → 글머리 기호 폰트 크기 14pt, 행간 1.4로 설정
  • 하단에 슬라이드 번호 플레이스홀더 추가: [삽입] → [머리글/바닥글] → 슬라이드 번호 체크
  • 슬라이드 번호 텍스트박스를 우측 하단으로 드래그해 위치 조정
💡 텍스트박스(직접 삽입)와 플레이스홀더는 다릅니다. 반드시 '플레이스홀더'를 사용해야 Claude in PPT가 내용을 인식할 수 있습니다.
4
레이아웃 3: 이미지+텍스트 혼합 슬라이드 만들기
  • 좌측 패널에서 세 번째 레이아웃 선택 → 이름 "이미지+텍스트"로 변경
  • 기존 플레이스홀더 모두 삭제 (클릭 후 Delete)
  • [슬라이드 마스터] 탭 → [플레이스홀더 삽입] → [그림] 선택
  • 좌측 영역(슬라이드 가로 50%)에 드래그해서 그림 플레이스홀더 배치
  • 다시 [플레이스홀더 삽입] → [콘텐츠] 선택 → 우측 영역에 텍스트 플레이스홀더 배치
  • 좌우 여백 각 0.5cm, 두 영역 사이 간격 0.5cm 유지
💡 [슬라이드 마스터] 탭은 슬라이드 마스터 편집 모드에서만 상단에 나타납니다. 일반 편집 화면에서는 보이지 않아요.
5
마스터 저장 및 적용 확인
  • [슬라이드 마스터] 탭 오른쪽 끝 [마스터 보기 닫기] 클릭
  • 일반 편집 화면으로 돌아오면, 슬라이드 우클릭 → [레이아웃] 메뉴 확인
  • "표지", "본문", "이미지+텍스트" 3가지 레이아웃이 목록에 있으면 성공
  • 각 레이아웃을 선택해 실제 슬라이드에 텍스트 입력 → 서식 자동 적용 확인
  • 파일 저장: Ctrl+S (또는 Cmd+S) → .pptx 형식으로 저장
💡 완성된 파일은 다음 과제(색 테마 추가, Claude in PPT 실습)에서 그대로 사용합니다. 파일명에 날짜를 붙여 저장해두세요.
실무 팁

마스터 먼저, 내용 나중

슬라이드 마스터를 완성하기 전에 내용을 채우면 서식이 틀어집니다. 항상 마스터 완성 후 내용을 작성하세요.

플레이스홀더 vs 텍스트박스

일반 텍스트박스가 아닌 플레이스홀더를 써야 Claude in PPT가 슬라이드 내용을 정확히 인식합니다.

테마 파일로 저장

[디자인] → [테마 저장] → .thmx 파일로 저장하면, 다른 PPT 파일에도 이 서식을 바로 불러올 수 있습니다.
