## 체크리스트

### 1. 전제조건 (Prerequisites)
- [ ] 필요한 프로그램이 모두 안내되어 있는가
- [ ] 필요한 계정/권한이 명시되어 있는가
- [ ] 사전 설정 단계가 빠짐없이 있는가

### 2. 용어 설명 (Terminology)
- [ ] 전문 용어 첫 등장 시 괄호 설명이 있는가
- [ ] 약어(API, CLI 등) 풀네임이 한 번은 나오는가

### 3. 단계 완결성 (Step Completeness)
- [ ] 이전 단계 → 다음 단계 전환이 자연스러운가
- [ ] 빠진 단계가 없는가 (중간에 "당연히 알 것"이라 생략한 부분)
- [ ] 각 단계에 최소 1장 이미지(지시서)가 있는가

### 4. 이미지-텍스트 정합 (Image-Text Alignment)
- [ ] 이미지가 바로 위/아래 텍스트와 일치하는가
- [ ] 이미지 내 강조 영역이 텍스트에서 설명하는 부분과 맞는가
- [ ] 이미지 해상도가 텍스트를 읽을 수 있을 만큼 충분한가

### 5. 조작 명확성 (Action Clarity)
- [ ] "어디를 클릭"이 구체적인가 (메뉴 > 하위메뉴 > 버튼)
- [ ] 입력할 텍스트가 정확히 제시되어 있는가
- [ ] 선택할 옵션이 명확히 지정되어 있는가

### 6. 결과 확인 (Verification)
- [ ] 각 단계 후 "이렇게 되면 성공" 기준이 있는가
- [ ] 최종 결과물이 어떤 모습인지 설명되어 있는가

### 7. 에러 대응 (Troubleshooting)
- [ ] 흔한 실수/에러에 대한 해결법이 있는가
- [ ] 권한 부족, 네트워크 오류 등 환경 문제 안내가 있는가

### 8. 이미지 품질 (Image Quality)
- [ ] 텍스트 가독성 (폰트 크기, 대비)
- [ ] 어노테이션 가시성 (화살표, 하이라이트 명확)
- [ ] 한국어 UI 텍스트 정확
