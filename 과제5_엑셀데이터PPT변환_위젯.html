Claude in PPT 엑셀 연동 보고서 자동화
과제 5 — 엑셀 데이터를
회사 서식 PPT 보고서로 변환
엑셀에 정리된 표와 수치 데이터를 복사해 Claude in PPT에게 전달하면, 과제 1~2에서 만든 회사 슬라이드 마스터 서식과 색 테마가 자동 적용된 PPT 보고서로 변환됩니다. 이 과제는 앞선 4개 과제의 결과물을 모두 활용하는 종합 실습입니다.
사전 준비
🛠

준비물

  • 과제 1~4에서 완성한 .pptx 파일 (마스터+색테마 적용)
  • 실습용 엑셀 파일 또는 샘플 표 데이터
  • Microsoft Excel (없으면 구글 스프레드시트 사용 가능)
✅

샘플 데이터 만들기

  • 엑셀 새 파일 열기 → A1부터 아래 데이터 입력:
  • 월 / 매출(만원) / 목표(만원) / 달성률(%)
  • 1월 / 1200 / 1000 / 120%
  • 2월 / 950 / 1000 / 95%
  • 3월 / 1350 / 1200 / 112%
단계별 실습
1
엑셀 데이터를 텍스트로 복사해 Claude에게 전달하기
  • 엑셀에서 표 전체 선택 (A1부터 데이터 끝까지) → Ctrl+C
  • PPT의 Claude 패널 입력창 클릭 → Ctrl+V로 데이터 붙여넣기
  • 붙여넣은 데이터 아래에 이어서 프롬프트 입력:
  • "위 데이터를 이용해 회사 서식이 적용된 PPT 보고서를 만들어줘. 슬라이드마다 표와 차트를 포함해줘."
  • Enter 전송
💡 엑셀 데이터를 그대로 붙여넣으면 탭으로 구분된 텍스트로 복사됩니다. Claude가 이를 자동으로 표로 인식합니다.
2
생성된 슬라이드에 마스터 서식이 적용됐는지 확인하기
  • Claude가 슬라이드를 자동 생성하면 각 슬라이드 확인
  • 슬라이드 우클릭 → [레이아웃] → 과제 1에서 만든 레이아웃이 적용됐는지 확인
  • 폰트와 색상이 과제 1~2의 마스터/테마와 일치하는지 확인
  • 일치하지 않으면: 해당 슬라이드 선택 → [홈] → [레이아웃] → 원하는 레이아웃 수동 적용
💡 Claude가 생성한 슬라이드가 마스터 서식을 완전히 따르지 않을 수 있습니다. 이런 경우 레이아웃을 수동으로 지정하면 됩니다.
3
차트 슬라이드 추가 요청하기
  • Claude 패널에 추가 요청 입력:
  • "매출과 목표를 비교하는 막대 차트 슬라이드를 하나 더 만들어줘. 회사 색 테마 적용해줘."
  • 생성된 차트 슬라이드 확인
  • 차트 클릭 → 우클릭 → [데이터 편집] → 실제 데이터가 맞게 들어갔는지 확인
  • 틀린 수치가 있으면 데이터 편집 창에서 직접 수정
💡 차트 수치가 잘못 들어간 경우 Claude에게 "차트 데이터를 다시 확인하고 수정해줘"라고 요청하거나 직접 수정하세요.
4
요약 분석 슬라이드 자동 생성하기
  • Claude 패널에 입력:
  • "이 데이터를 분석해서 핵심 인사이트 3가지를 요약한 슬라이드를 만들어줘"
  • Claude가 데이터 패턴을 분석해 자동으로 요약 슬라이드 생성
  • 생성된 내용이 실제 데이터와 일치하는지 수치 검토
  • 인사이트 문구를 더 구체적으로 수정하고 싶으면 직접 텍스트 편집
💡 AI가 생성한 인사이트는 반드시 사람이 검토해야 합니다. 수치 해석이 실제 상황과 다를 수 있으니 꼭 확인하세요.
5
전체 보고서 검토 후 최종 저장하기
  • F5로 슬라이드쇼 실행 → 처음부터 끝까지 검토
  • 확인 항목: ① 회사 로고/색상 일관 적용 ② 수치 정확성 ③ 오탈자 없음 ④ 슬라이드 흐름
  • 수정 사항은 Claude 패널 또는 직접 편집으로 처리
  • 완료 후 Ctrl+S 저장
  • 추가로 PDF 내보내기: [파일] → [내보내기] → [PDF/XPS 만들기] → 저장
💡 PDF로도 저장해두면 폰트/레이아웃이 깨지지 않아 상사나 고객에게 공유하기 좋습니다.
실무 팁

데이터 먼저 정리

엑셀 데이터를 Claude에게 넘기기 전에 빈 행/열, 합계 행을 깔끔하게 정리해두면 훨씬 정확한 슬라이드가 생성됩니다.

수치는 반드시 검토

AI가 생성한 차트와 인사이트의 수치는 반드시 원본 엑셀과 대조해 검토하세요. 자동화와 검토는 함께 합니다.

PDF 최종 배포

완성된 보고서는 PDF로 내보내기 하면 어떤 기기에서 열어도 서식이 그대로 유지됩니다.
